Co-op with Tulip Retail

Introduction

In the Summer of 2017, I worked as a full-stack developer at Tulip Retail, a software company that builds retail and ecommerce software.

Job Description

This was my second co-op term with Tulip Retail. For this term, I moved to commerce engine platform team. Our team was responsible for maintenance and upgrades to the core back-end platform used across all of the Tulip's products.

Goals

Improve Back-End Development Skills

Most of my time over this four-month term was spent on working on the back-end systems for Tulip's core platform. I learned a lot about how to build scalable back-end systems and about the technologies that can be used to build these systems. Some of the technologies that were new to me included:

  • Docker
  • Solr
  • Beanstalk
  • Codeception

On top of these technologies, I learned a lot about how to architect and structure large software systems.

Improve Project Management Skills

Over the course of the work term, I gained an increasing amount of autonomy to manage and work on projects independently. I would typically spend between 2 and 4 weeks on each project, determining what needs to be done, how to do it, and ultimately implementing the project. By working with this autonomy and managing my own tasks, my project and time management skills grew.

Successfully Deliver Projects and Features

The most important part of any job is to get work done and to be an asset to the company. This was my main focus for the work term, delivering high quality software on time and on spec. In this respect, the term was a success; I was able to work with my team and across teams to deliver the necessary software to ship new features and to enable our clients. Over the course of four months, I was able add many new features to the core product, on top of working on some larger projects including:

  • Upgrading the back-end from PHP 5.3 to PHP 7.
  • Creating a new API documentation framework and engine.
  • Refactoring and redesigning a core API.
  • Using a new framework to build a test suite for a core project.

Acknowledgements
