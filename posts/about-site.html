This site

This site was designed as a simple, frameworkless, single page application. The reason I used an SPA rather than a more traditional application was because I felt like experimenting on a smaller scale (this) before I began a larger SPA project.

It is hosted through Github Pages.

Inspiration for the hashchanging technique on this site came from here which allows the URL to dictate the state of the page even though it is an SPA.

This site isn't responsive yet, but will be soon.

My career is still young, so this website is very small also. I hope to expand it as I do more interesting things worth mentioning.
