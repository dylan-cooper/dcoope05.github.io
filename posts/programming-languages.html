Programming Languages

This is a (somewhat) brief overview of the programming languages and technologies that I have used. I will try to keep it up to date as much as possible.

JavaScript is the language of the web, and has consequently become my primary programming language. I am comfortable working with vanilla JavaScript when there is a need, but also familiar with many languages, frameworks and technologies surrounding the language including:

  • JQuery
  • Node.js
  • npm
  • electron
  • knockout
  • React
  • Redux
  • Express
  • webgl
  • Sencha Ext JS

C

C is the primary language used in University of Guelph computing courses. I have a thorough understanding of the ins and outs of the language. I've used it for multiple programs including a maze solver, binary search tree, lexer/parser and sudoku puzzle solver.

C# and .NET MVC

I used the C# programming language with the .NET MVC framework during my first co-op term at the University of Guelph. I became very familiar with the framework and language, as part of my job was introducing it to other members of the team who would be working on the system in the future.

Java

My experience in Java is limited to classes in object oriented design at the University of Guelph. The most common use case for Java in these courses was Swing-based GUI programs, so I have used it to make a few of those. It is also my language of choice for programming contests and Project Euler questions.

QBASIC

QBASIC was the first programming language I learned, in high school. As this was when I was just learning the basics of programming, the programs that I wrote were pretty "basic". I leave it on this list because this is really where I fell in love with programming. Below is some old code I found from an exercise which was before the concept of nested loops was introduced. 🙂 

1 ' Description : Prints lines of Stars
2 ' Programmer : Dylan Cooper
3 ' Date : Feb 17

4 ' Variables
5 '     I - controls the loop
6 '     NUMSTARS - number of lines of stars
7 '     STARCOUNT - number of stars already on that line

50 CLS
70 PRINT "How many lines of stars would you like?";
80 INPUT NUMSTARS
90 FOR I = 1 TO NUMSTARS
100     LET STARCOUNT = STARCOUNT + 1
110     IF STARCOUNT < I THEN PRINT "*";  ELSE PRINT "*"
120     IF STARCOUNT < I THEN GOTO 100
130     LET STARCOUNT = 0
140 NEXT I
